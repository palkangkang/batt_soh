# 双层闭环受控实验方案（Top3+对照）

## 0. 方案摘要
- 生成时间：2026-04-08 12:41:40
- 统一标签区间：`0.3 <= q_discharge <= 1.3`，战略层与战术层同口径。
- 历史样本规模：`n_rows=102135`，`n_clusters=180`。
- 战略层建议倍率上限：`3.50C`。
- 试验结构：`1个对照组 + 3个干预臂`。

## 1. 研究假设
- H0（零假设）：在战略约束内实施区间配时替代后，`q_discharge_{t+1}` 的均值与对照组无差异。
- H1（研究假设）：至少一个干预臂可显著改善 `q_discharge_{t+1}`，且不提升安全风险。

## 2. 随机化与分层设计
- 随机化单元：`policy + cell_code`（整电芯级随机化，避免同电芯跨组污染）。
- 分层变量：policy三元参数、`is_abnormal_cell`、基线容量分位。
- 分组结构：对照组维持当前策略；干预组在同一战略约束内执行指定区间替代。

## 3. 干预实施规则（Top3 + 对照）
- Phase A（战略层）：先统一执行倍率上限策略，验证风险净改善。
- Phase A执行阈值：最大充电倍率 `< 3.50C`。
- Phase B（战术层）：在Phase A约束内实施 Top3 区间 `±5pp` 替代。

## 4. 终点定义与统计分析计划
- 主要终点：`q_discharge_{t+1}`（下一循环放电容量，单位Ah）。
- 次要终点：30-cycle 衰减斜率、`dt_s>3600` 事件率、温度越界率。
- 分析口径：ITT（意向治疗）为主，PP（依方案）为辅。
- 统计方法：组间均值差 + 聚类稳健标准误（聚类单元 `policy+cell`）。
- 显著性控制：主终点按 `p<0.05`，多臂比较同步报告 FDR 修正后的 `q`。

## 5. 样本量与观察窗建议
- 建议每臂最少 `20` 个电芯簇（clusters），优先保证分层后每层>=5个簇。
- 建议观察窗口：最短 `30 cycles`，推荐 `60 cycles` 以降低短期波动。
- 若阶段中期效果绝对值 < `0.0002Ah/5pp` 且 CI 长期跨0，进入“延长观察或停臂复盘”。

## 6. 停止规则与安全阈值
- 连续3个 cycle 出现容量显著恶化且伴随安全告警（温度/电流）时，立即停臂。
- 任一组温度保护或电流保护触发率较对照组上升超过 `20%`，暂停该组并回滚。
- 如发生策略冲突（超出倍率上限），该批次数据标记为协议违背并纳入PP剔除。

## 7. 上线闸门（Go/No-Go）
- Go：主要终点改善显著（q<0.10）且安全指标不劣于对照组。
- Hold：效果方向正确但CI跨0，继续扩样验证。
- No-Go：效果为负或出现明确安全恶化。

## 8. 实施检查清单
- 每cycle记录字段：`policy`、`cell_code`、`cycles`、`q_discharge`、`cross_bin_inc_01..60_h`、`is_abnormal_cell`。
- 每日执行核查：分组样本数、协议违背率、温度/电流越界率、缺失值比例。
- 每周复盘：ITT/PP差异、各干预臂效应方向稳定性、是否触发停臂规则。
