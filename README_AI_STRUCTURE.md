# Python AI训练/测试参考目录（简版）

- data/raw: 原始数据
- data/processed: 处理后数据
- data/sample: 小样本调试数据
- src/models: 模型定义
- src/utils: 工具函数（如数据加载）
- train/train.py: 训练入口
- test/test.py: 测试/评估入口
- configs/train.yaml: 训练配置
- outputs/checkpoints: 模型权重
- outputs/logs: 训练日志
- outputs/predictions: 预测输出
- notebooks: 实验记录
- scripts/run_train.ps1: 训练脚本
- scripts/run_test.ps1: 测试脚本
